# 📚 EduSolve PRO
> **Encuentra respuestas de Santillana 3° de forma dinámica y eficiente.**

EduSolve PRO es una plataforma educativa desarrollada por **Alonixz-Group** diseñada para ayudar a estudiantes de secundaria a gestionar y encontrar recursos académicos de manera rápida, utilizando una interfaz minimalista y tecnología en tiempo real.

---

## 🚀 Características Principales
* 🔍 **Buscador Dinámico:** Filtra respuestas y recursos de Santillana 3° de secundaria.
* ⚡ **Interfaz Flat Design:** Diseño limpio, sin distracciones, optimizado para la concentración.
* 📱 **Totalmente Responsivo:** Acceso fluido desde celulares, tablets y computadoras.
* 💾 **Integración con Firebase:** Base de datos en tiempo real para actualizaciones constantes.

## 🛠️ Stack Tecnológico
* **Frontend:** HTML5, CSS3 (Custom Flat Design), JavaScript (Vanilla).
* **Backend/Database:** Firebase Realtime Database.
* **Seguridad:** Sistema de validación de usuarios y analizador de errores con delay de 5.2s.

## 🛡️ Sistema de Gestión de Errores
La plataforma cuenta con un **Embudo de Errores** avanzado que garantiza la estabilidad:
1.  **Limpiador (5.1s):** Filtra ruidos de carga (como el triple inicio de Firebase) y errores de sintaxis conocidos.
2.  **Analizador (5.2s):** Valida la integridad de la sesión y, en caso de fallo crítico, redirige a una página de error personalizada.
3.  **Interfaz de Emergencia:** Página `error.html` con estética amigable y soporte técnico directo.

### 🔧 Instalación y Despliegue
1. Clona el repositorio.
2. Configura tus credenciales de Firebase en el script principal.
3. Abre `index.html` en tu servidor local o despliega en Firebase Hosting.
